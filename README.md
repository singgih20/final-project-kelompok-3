**Final project bootcamp online sanbercode - Kelompok 3.**  
Kelas : Laravel lanjutan  
Study case : Toko buku online

**Steps**  
- copy project ini menggunakan `git clone`  
- lakukan `composer install` untuk mendownload dependency  
- buat database, copy paste .env.example menjadi .env dan atur database, pusher & generate key. setelah itu jalankan `php artisan migrate`  
- jalankan `npm install && npm run dev` untuk mendownload vue dari node_modules
